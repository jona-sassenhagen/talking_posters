Write a framework that supports generating "talking posters".

Step one is to receive a prompt.

You (codex) will then generate an infographic based on this prompt.

Then, you are to read the infographic, and convert it into a web app/website.

Any section on the web app should be clickable; upon the click, the text in that section should be read out (using `say -v "Petra (Premium)"` to generate audio).
